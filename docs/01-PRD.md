# 01 - 产品需求文档（PRD）

> 文档版本：v1.0  
> 目标产品：掘金风格技术博客社区（企业级）

---

## 1. 产品定位

打造一个面向技术人群的企业级博客社区平台，以内容创作、技术交流、知识变现为核心。平台风格参考掘金（Juejin），强调**高质量内容、精准推荐、创作者友好**。

---

## 2. 核心功能模块

### 2.1 内容中心（P0）

| 功能 | 描述 |
|------|------|
| 文章发布 | 支持 Markdown / 富文本双模式，实时预览 |
| 草稿箱 | 自动保存草稿，支持多草稿管理 |
| 历史版本 | 文章修改记录，支持版本回滚（预留） |
| 标签体系 | 文章可打多个标签，标签有热度与关注数 |
| 专栏 | 创作者可创建专栏，系列文章聚合 |
| 内容审核 | 机器预审核 + 人工复核，状态机驱动 |

### 2.2 Feed 流（P0）

| 流类型 | 排序逻辑 |
|--------|----------|
| 推荐流 | 基于用户阅读/点赞/收藏行为的协同过滤 + 热度加权 |
| 关注流 | 按时间倒序展示关注作者的新文章 |
| 热榜 | 近 24h / 7d / 30d 综合热度排序 |
| 最新发布 | 严格时间倒序 |

### 2.3 用户系统（P0）

| 功能 | 描述 |
|------|------|
| 注册/登录 | 邮箱 + 密码、GitHub OAuth、Google OAuth |
| 个人主页 | 展示文章、动态、关注/粉丝、获赞数 |
| 关注体系 | 单向关注，大 V 认证标识 |
| 等级勋章 | 基于创作量、互动量、阅读量的成长体系（预留） |

### 2.4 互动体系（P0）

| 功能 | 描述 |
|------|------|
| 点赞 | 文章/评论均可点赞，防刷限流 |
| 收藏 | 支持收藏夹分类管理 |
| 评论 | 嵌套回复（二级评论），支持点赞 |
| 阅读统计 | 去重 UV / PV，创作者可见 |

### 2.5 搜索发现（P1）

| 功能 | 描述 |
|------|------|
| 全文搜索 | 基于 Elasticsearch，支持标题/摘要/标签/作者搜索 |
| 热搜词 | 近 1 小时搜索量 Top10 |
| 相关推荐 | 文章底部基于标签相似度的推荐 |

### 2.6 创作者中心（P1）

| 功能 | 描述 |
|------|------|
| 数据分析 | 阅读量/点赞/收藏/粉丝增长趋势图 |
| 内容管理 | 已发布/草稿/审核中/被驳回文章管理 |
| 收益管理 | 付费专栏销售数据、打赏记录（预留） |

### 2.7 运营管理（P1）

| 功能 | 描述 |
|------|------|
| Banner 管理 | 首页轮播图配置 |
| 公告系统 | 全站公告与弹窗 |
| 举报反垃圾 | 用户举报入口 + 内容安全策略 |
| 内容审核后台 | 审核员对文章进行通过/驳回操作 |

### 2.8 商业化（P2 - 预留）

- 付费专栏
- 文章打赏
- 广告投放位

---

## 3. 非功能性需求（企业级标准）

| 维度 | 目标 |
|------|------|
| **性能** | 首屏 LCP < 2.5s，接口 P99 < 200ms |
| **可用性** | 全年 SLA 99.9%，核心服务具备降级能力 |
| **扩展性** | 微服务架构，支持水平扩展；数据库预留分库分表 |
| **安全性** | JWT + RefreshToken、接口限流、XSS/CSRF 防护、内容安全审核 |
| **可观测性** | 全链路追踪、结构化日志、指标监控、自动告警 |
| **SEO** | 文章页 SSR，结构化数据，语义化 URL |

---

## 4. 用户角色

| 角色 | 权限描述 |
|------|----------|
| 访客 | 浏览文章、搜索、查看用户主页 |
| 注册用户 | 发布文章、评论、点赞、收藏、关注 |
| 创作者 | 进入创作者中心查看数据、创建专栏 |
| 审核员 | 审核文章、处理举报 |
| 管理员 | 运营配置、用户管理、全权限 |
